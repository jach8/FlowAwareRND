\section{Methodology}
\subsection{Heston Model \& Characteristic Function}
SDEs:
\begin{align*}
dS_t &= r S_t dt + \sqrt{v_t} S_t dW_t^S \\
dv_t &= \kappa (\theta - v_t) dt + \xi \sqrt{v_t} dW_t^v, \quad \langle dW^S, dW^v \rangle = \rho dt
\end{align*}

Closed-form characteristic function used for pricing via damped P1/P2 Fourier inversion.

\subsection{Multi-Expiry Calibration}
Minimize weighted IV RMSE across expiries, weighting shorter maturities higher ($1/\tau$) and incorporating |GEX| + |charm| for strike prioritization.

\subsection{PDF Extraction}
Apply Breeden-Litzenberger: $f_Q(K,\tau) = e^{r\tau} \frac{\partial^2 C}{\partial K^2}$, smoothed via UnivariateSpline.
